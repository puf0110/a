1. **What I have implemented**

-   User login with no password / 15 mins
-   Plan view with showing construction plan on canvas with task markers / 1 hour
-   Add task with coordination, task title and checklist / 15 mins
-   View Task list and edit tasks / 1 hour
-   RxDB Dexie with IndexedDB to store user, task, plan for offline functionality / 30 mins
-   Syncing every 8 seconds with custom sync logic / 1 hour
-   Robust error handling and validation / 1 hour
-   Display only visible markers on the plan / 15 mins
-   Soft deletes of tasks / 15 mins

    Overall 4h 30min
